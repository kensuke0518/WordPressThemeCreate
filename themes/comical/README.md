# comicalテーマ
技術系のことを記録するサイトのテーマ

## ショートコード
### コードをハイライトする
`[highlight]ここにコードを入力する[/highlight]`  
  
### 見出し
`[heading hnum='NUMBER' num='NUMBER' mt='NUMBER']`
- hnum
見出しの大きさを表します。  
<h1>にしたければ 1 を、<h2>にしたければ 2 を入れてください。  
- num
見出しのスタイルを変更します。
- mt
見出しの上に余白（マージン)をつけます。マージンの大きさは1〜100pxまで
